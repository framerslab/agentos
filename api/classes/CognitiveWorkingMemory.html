Class CognitiveWorkingMemory

Cognitive working memory with Baddeley-inspired slot management.

Implements IWorkingMemory for the basic key-value surface (get/set/delete/clear/has/size/getAll), with slot management (phonological loop, visuospatial sketchpad, episodic buffer) layered on top.
