Class ConsolidationLoop

Self-improving background consolidation loop with 6 ordered steps: prune, merge, strengthen, derive, compact, re-index.

All database operations use the async StorageAdapter API through the shared Brain connection. The run() method is async to accommodate both the database calls and the LLM-backed derive step.
