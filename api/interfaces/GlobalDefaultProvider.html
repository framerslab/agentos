Interface GlobalDefaultProvider

Shape of the global default-provider config.

All fields are optional. Setting just provider lets AgentOS pick the provider's default model for each task type; setting apiKey / baseUrl provides credentials without touching the environment.

interface GlobalDefaultProvider {
    provider?: string;
    model?: string;
    apiKey?: string;
    baseUrl?: string;
}
