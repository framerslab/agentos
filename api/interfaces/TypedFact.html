Interface TypedFact

A typed fact in the Hindsight memory schema. Carries narrative text, embedding, temporal envelope, participants, reasoning markers, extracted entities, and a confidence score in [0, 1]. Confidence defaults to 1.0 for World/Experience/Observation facts; the Opinion bank stores (text, confidence, timestamp) tuples per §2.2.

interface TypedFact {

}
