Interface ICodeSandbox

Interface for the Code Execution Sandbox.

Example

const sandbox = new CodeSandbox();

const result = await sandbox.execute({
  language: 'python',
  code: `
    import json
    data = {"result": 42}
    print(json.dumps(data))
  `,
});

console.log(result.output?.stdout); // '{"result": 42}'
