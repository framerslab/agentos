# Support

Need help with AgentOS?

- **Documentation**: [docs.agentos.sh](https://docs.agentos.sh)
- **Questions and discussion**: [Discord](https://wilds.ai/discord)
- **Bugs and feature requests**: [GitHub Issues](https://github.com/framerslab/agentos/issues)
- **Security issues**: see the [Security Policy](.github/SECURITY.md). Do not open a public issue for vulnerabilities.
- **Commercial, partnership, or sponsorship inquiries**: team@frame.dev (see [SPONSORS.md](SPONSORS.md))

Before opening an issue, search existing issues and check the documentation. For how to contribute, see [CONTRIBUTING.md](CONTRIBUTING.md).
